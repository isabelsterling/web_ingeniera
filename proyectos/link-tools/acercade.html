¿Qué es LinkKit?

LinkKit es una herramienta web gratuita que ofrece un conjunto de utilidades para trabajar con enlaces y URLs. Actualmente proporciona 9 herramientas en una sola interfaz:

  • ✂️ Acortador de URLs — Convierte enlaces largos en cortos
  • ⬛ Generador de Códigos QR — Crea códigos QR descargables
  • ↩️ Expansor de Enlaces — Descubre la URL real detrás de enlaces acortados
  • ✅ Verificador de Enlaces — Comprueba si un enlace está activo o roto
  • 👁️ Preview Redes Sociales — Previsualiza enlaces como en Twitter/WhatsApp
  • 📊 Constructor UTM — Crea URLs con parámetros de seguimiento
  • 🔤 Codificador URL — Codifica/decodifica caracteres especiales
  • 🧹 Limpiador de Enlaces — Elimina parámetros de seguimiento
  • 📷 Lector de Códigos QR — Lee QR desde imágenes

Tecnologías Utilizadas

LinkKit está construido con tecnologías web estándar:

  • HTML5, CSS3, JavaScript (Estado original)
  • QRCode.js — Generación de códigos QR
  • jsQR — Lectura de códigos QR
  • is.gd API — Servicio de acortamiento de URLs
  • All Origins API — Proxy para solicitudes CORS
